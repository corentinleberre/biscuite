## Tech Stack

**Client:** React 18, TailwindCSS

The game is deployed on Vercel.

## Run Locally

Clone the project

```bash
  git clone https://github.com/corentinleberre/biscuite
```

Go to the project directory

```bash
  cd biscuite
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
